# 🛒 MERN Login Register Dashboard (JWT + Bcrypt)

## 📌 Project Overview

This is a simple full-stack project built using **React (Frontend)** and **Express.js (Backend)**.
It demonstrates a complete authentication system including **Register, Login, and Dashboard access using JWT authentication**.

---

## 🚀 Features Implemented

* 🔐 User Registration (with password hashing using Bcrypt)
* 🔑 User Login (JWT token generation)
* 🛡️ Protected Dashboard Route
* 🔄 Authentication using Middleware
* 📦 REST API using Express.js
* ⚛️ React Frontend with Routing
* 💾 In-memory data storage (No DB for learning purpose)
